# Dependencies  

## You should install  
python3 (tested on 3.13.7)  
python3-venv  
python3-pip (tested on 25.3)  

## Automatically installed  
pygame-ce (tested on 2.5.6)  
Pillow    (tested on 12.0.0)  

# Installation  

## Linux  
1. Clone this project into your install directory.  
2. Run `setup.sh`.  
3. Create a shortcut to `run.sh`.  

## Windows  
1. Clone this project into your install directory.  
2. Run `setup.ps1`.  
3. Create a shortcut to `run.ps1`.  

# Usage  

1. Activate the shortcut.  
2. Move your mouse.  
3. Tap to close the program again.  

## Notes  

Because the program starts up a virtual environment every time it is ran, it starts very slowly.  

# Deinstallation  

1. Remove the install directory.  
2. Remove the shortcut.
